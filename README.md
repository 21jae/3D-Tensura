## 게임 소개
이 게임은 "전생했더니 슬라임이었던 건에 대하여" 라는 일본 라이트 노벨을 원작으로 했으며, 주인공이 슬라임으로 전생하여 다양한 모험을 겪는 내용입니다. 
이를 게임속에서 원작의 느낌을 즐길수있도록 개발하였습니다.</br>


## * **게임 플레이 특징**
